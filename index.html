Introdução

Este programa foi feito na linguagem Python e foi criado para auxiliar alunos da disciplina matemática discreta.

O programa abrange o assunto de conjuntos, união, interseção, diferença e complementar entre dois conjuntos.

Os materiais e fontes utilizados para a construção do algoritmo:

  • Python 3.6;
  • Editor de Texto Atom;
  • Windows PowerShell;
  • Livro: Fundamentos de Matemática Elementar Vol.1 Conjuntos e Funções Gelson Iezzi e Carlos Murakami e
  • Learn Python the Hard Way Zed A. Shaw.

Declarando as variáveis

Primeiro devemos declarar a variável em formato de lista que irá receber os elementos do conjunto A:

a = []

E a variável que irá receber o conjunto B:

b = []

Foi escolhido a variável em formato de lista pois, tratando-se de conjuntos é a melhor para trabalhar.


Primeiro Laço

Nesse algoritmo iremos utilizar a estrutura de laço para receber quantos elementos o usuário quiser informar.

while True:
elemento_a = int(input("Digite o(s) elemento(s) do conjunto A, sendo x inteiro e positivo: "))

Para finalizar o laço iremos utilizar um comando:

if elemento_a == -1:
break

Dessa forma o laço para, quando o usuário informar inserir o valor "-1."

Para salvar cada elemento que o usuário digitar utilizamos uma função de lista o .append(), ele salva uma variável dentro de uma lista.

a.append(elemento_a)

Para o segundo laço utilizamos o mesmo procedimento, apenas trocando a variável "a" por "b."
